# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this repository, please do not open a public issue with exploit details.

Instead:

1. Prepare a short description of the issue
2. Include reproduction steps and impact
3. Share it privately through the contact path listed on [postplus.io](https://postplus.io)

## Scope

This repository is primarily a content and metadata repository for installable skills.

Relevant issues may include:

- malicious or unsafe instructions
- unsafe default commands or workflows
- secrets or sensitive data accidentally committed
- harmful external links or supply-chain references
- metadata or packaging issues that could cause unsafe installation behavior

## What To Include

Please include:

- affected file paths
- reproduction steps
- expected behavior
- observed behavior
- severity or likely impact

## Disclosure

We will review reports as quickly as practical and aim to coordinate disclosure responsibly.
